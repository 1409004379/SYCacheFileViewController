# SYCacheFileViewController
缓存文件视图控制器
#### 一级视图控制器：缓存文件目录列表
#### 二级视图控制器：缓存文件列表
#### 缓存操作：
###### 缓存目录操作
* 删除：删除目录（包括目录下的子目录及文件）
###### 缓存文件操作
* 查看：视频播放、音频播放、文档查看、图片查看……
* 删除：文件
